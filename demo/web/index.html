Bat Web Demo

This demo uses Bat for Web and the WebVoiceProcessor to:

  1. Create an instance of Bat with the default model file.
  2. Acquire microphone (& ask permission) data stream and convert to voice processing format (16kHz 16-bit linear PCM). The downsampled audio is forwarded to the Bat engine. The audio does not leave the browser: all processing is occurring via the Bat WebAssembly code.
  3. Listen for language detection scores from the Bat engine and output them to the page.
After entering the AccessKey, click the "Start Bat" button.
